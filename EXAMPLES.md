# 使用示例

## 快速开始

### 1. 环境准备

```bash
# 克隆项目
git clone https://github.com/yourusername/renko-short-strategy.git
cd renko-short-strategy

# 创建环境
conda create -n renko python=3.11 -y
conda activate renko
pip install -r requirements.txt
```

### 2. 配置 Token

```powershell
# Windows
[Environment]::SetEnvironmentVariable("TUSHARE_TOKEN", "your_token", "User")
```

## 每日运行流程

```bash
# 收盘后运行
python main.py run

# 查看选股结果
cat data/output/selection_20260409.csv
```

## 分步执行

```bash
# 只更新数据
python main.py update

# 只构建信号池
python main.py build

# 只训练权重
python main.py train

# 只选股
python main.py select --top 10
```

## 参数说明

| 参数 | 说明 | 示例 |
|:-----|:-----|:-----|
| `--top` | 选股数量 | `--top 10` |
| `--max-stocks` | 更新股票数量限制 | `--max-stocks 100` |
| `--skip-update` | 跳过数据更新 | `--skip-update` |
| `--skip-build` | 跳过构建信号池 | `--skip-build` |
| `--skip-train` | 跳过训练 | `--skip-train` |
